# Вариант №4
# Сервис управления разрешениями (Permission Service)

## Сущность: Permission (Разрешение)

Тонкая настройка прав: кто может редактировать расписание, кто только смотреть, кто может назначать замены.

---

### 1. Информация для создания сущности

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|-----------|----------------|-----|-------------|-----------------------|
| `name` | Название разрешения | Да | str | длина ≤ 100, уникально | — |
| `resource` | Ресурс, к которому применяется разрешение (например: schedule, group, report) | Да | str | длина ≤ 100 | — |
| `action` | Действие над ресурсом | Да | str | длина ≤ 50 (create, read, update, delete, manage) | — |
| `description` | Описание разрешения | Нет | str | длина ≤ 255 | `''` |

**Уникальные комбинации параметров:**
- `name`
- `resource` + `action`

### 2. Информация, возвращаемая при успешном создании

| Параметр | Тип |
|----------|-----|
| `id` | int |
| `name` | str |
| `resource` | str |
| `action` | str |
| `description` | str |

---

## Изменить сущность по ID

### 3. Информация для изменения сущности

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|-----------|----------------|-----|-------------|-----------------------|
| `name` | Новое название разрешения | Нет | str | длина ≤ 100, уникально | текущее значение |
| `description` | Новое описание разрешения | Нет | str | длина ≤ 255 | текущее значение |

**Примечание:** Параметры `resource` и `action` не могут быть изменены после создания сущности.

### 4. Информация, возвращаемая при успешном изменении

| Параметр | Тип |
|----------|-----|
| `id` | int |
| `name` | str |
| `resource` | str |
| `action` | str |
| `description` | str |

---

## Удалить сущность по ID

Вернет `True` (удалено) или `False` (не найдено) в поле `deleted`.

### Информация, возвращаемая при удалении

| Параметр | Тип | Пояснение |
|----------|-----|-----------|
| `deleted` | bool | `true` - если сущность была удалена, `false` - если сущность не найдена |
